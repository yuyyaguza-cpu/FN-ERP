# 🚀 คู่มือติดตั้งระบบจัดการบริษัท (Supabase)
ไม่ต้องติดตั้งอะไรเพิ่มเติม เปิดใช้งานได้บนเบราว์เซอร์ทันที

---

## ขั้นตอนที่ 1: สร้างบัญชี Supabase (ฟรี)

1. เปิดเบราว์เซอร์ไปที่ **https://supabase.com**
2. กด **"Start your project"** → Sign up ด้วย GitHub หรือ Email
3. กด **"New project"**
4. ตั้งชื่อโปรเจค เช่น `company-system`
5. ตั้งรหัสผ่าน Database (จำไว้หรือจดไว้)
6. เลือก Region ที่ใกล้ที่สุด เช่น `Southeast Asia (Singapore)`
7. กด **"Create new project"** → รอประมาณ 1-2 นาที

---

## ขั้นตอนที่ 2: รัน SQL สร้างตาราง

1. เมื่อ Project พร้อม ไปที่แถบซ้าย → **SQL Editor**
2. กด **"New query"**
3. เปิดไฟล์ **`supabase_setup.sql`** ด้วย Notepad
4. **คัดลอกทั้งหมด** (Ctrl+A → Ctrl+C)
5. วางในช่อง SQL Editor (Ctrl+V)
6. กด **"Run"** (หรือ Ctrl+Enter)
7. ตรวจสอบว่าด้านล่างแสดง **"Success"**

---

## ขั้นตอนที่ 3: คัดลอก URL และ Key

1. ไปที่แถบซ้าย → **Settings → API**
2. คัดลอก **"Project URL"** (รูปแบบ: `https://xxxxxxxx.supabase.co`)
3. คัดลอก **"anon public"** key (ข้อความยาวๆ ขึ้นต้นด้วย `eyJ...`)

---

## ขั้นตอนที่ 4: เปิดใช้งานแอป

1. เปิดไฟล์ **`app.html`** ด้วยเบราว์เซอร์ (Double-click)
2. ใส่ **Project URL** และ **anon key** ที่คัดลอกมา
3. กด **"เชื่อมต่อและเริ่มใช้งาน"**
4. ระบบจะโหลดและพร้อมใช้งาน! 🎉

---

## ขั้นตอนที่ 5: แชร์ให้ทีม (ทางเลือก)

### วิธีที่ 1: ส่งไฟล์ app.html
- ส่งไฟล์ `app.html` และ `supabase_setup.sql` ให้เพื่อนร่วมงาน
- ให้เพื่อนร่วมงานเปิด `app.html` และใส่ URL + Key เดียวกัน
- ทุกคนจะเห็นข้อมูลเดียวกัน Realtime!

### วิธีที่ 2: Host บน Netlify (ฟรี — เข้าผ่าน URL ได้เลย)
1. ไปที่ https://netlify.com → Sign up ฟรี
2. ลากไฟล์ `app.html` ไปวางในหน้า Deploy
3. ได้ URL เช่น `https://company-xyz.netlify.app`
4. แชร์ URL นี้ให้ทุกคนในทีม!

## ฟีเจอร์ทั้งหมด

| ฟีเจอร์ | รายละเอียด |
|---------|-----------|
| 📊 Dashboard | สถิติภาพรวม + งานและผลงานล่าสุด |
| ✅ Todo List | เพิ่มงาน มอบหมาย ตั้งความสำคัญ กำหนดส่ง |
| 📁 ผลงาน | อัพโหลดรูปภาพและเอกสาร preview ได้ |
| 👥 พนักงาน | จัดการรายชื่อ ดูสถิติงานค้าง |
| 🔔 แจ้งเตือน | Realtime ทุกคนเห็นพร้อมกัน |
| 🔐 ข้อมูล | เก็บบน Supabase (Cloud) ปลอดภัย |

---

## หมายเหตุ

- Supabase ฟรี tier รองรับ **500MB database** และ **1GB storage** เพียงพอสำหรับทีมเล็ก-กลาง
- ข้อมูล URL และ Key ถูกเก็บเฉพาะในเบราว์เซอร์ของแต่ละคน ไม่ส่งไปที่อื่น
- หากต้องการ Reset ให้กดปุ่ม "เปลี่ยน Supabase Project" ที่หน้า Login
