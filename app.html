1ไปที่ supabase.com → สร้าง Project ฟรี
2ไปที่ SQL Editor → วางโค้ดจากไฟล์ supabase_setup.sql → กด Run
3ไปที่ Settings → API → คัดลอก URL และ anon key มาใส่ด้านล่าง
